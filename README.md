📚 Vocab App

Приложение для изучения иностранных слов с использованием интервальных повторений и игровых механик. Делал его как удобный и не скучный способ учить слова — без зубрёжки, но с прогрессом.

🚀 Основной функционал
📖 Обучение
  Система карточек — обычное изучение слов через перевод
  Интервальные повторения — чтобы слова реально запоминались
🎮 Игровые режимы
  Match (сопоставление)
    даётся 5 слов на английском
    и их переводы
    нужно правильно сопоставить пары
    после правильного выбора пара исчезает
  Битва с боссом
    правильный ответ → наносишь урон боссу
    ошибка → получаешь урон
    по сути проверка знаний в игровом формате
📊 Статистика
  Общая активность
    Heatmap за последние 126 дней
    чем больше активности — тем ярче день
    
🎨 Темы

  Можно менять внешний вид приложения:
    Emerald
    Amethyst
    Ocean
    Cyberpunk
    
🧠 Алгоритмы
⏳ Интервальные повторения

Система уровней:

  Level 0 — новое слово (сразу доступно)
  Level 1 - через 24 часа 
  Level 2 — через 1 неделю
  Level 3 — почти выучено через 1 месяц
    После этого слово считается выученным и слова будет появлятся только через 90 дней
    если отвечаешь правильно → уровень растёт
    если ошибся → сбрасывается
    
⚔️ Логика битвы
  Берутся слова, у которых наступил nextReview
  Из них случайно выбирается примерно 50%
  Формируется текущая игровая сессия

Это нужно, чтобы:

  не перегружать пользователя
  и равномерно распределять обучение
  
🔥 Heatmap
  Показывает последние 126 дней
  Работает как скользящее окно
  Чем больше действий в день — тем ярче клетка
  
🏗 Структура проекта (Frontend)

app/(tabs)/

_layout.tsx     — навигация и темы

index.tsx       — главная логика (карточки слов)

battle.tsx      — режим битвы с боссом (HP, логика боя)

matching.tsx    — режим сопоставления слов (Match)

stats.tsx       — статистика и heatmap

setting.tsx     — настройки:
                  - добавление слов
                  - изменение тем
                  - очистка прогресса
src/

store/useStore.ts   — глобальное состояние (Zustand)

api/api.ts          — работа с API
